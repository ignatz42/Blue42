# Get-B42Globals

## SYNOPSIS
Retrieves the current set of globals parameters and their values.

## SYNTAX

```
Get-B42Globals [<CommonParameters>]
```

## DESCRIPTION
The Get-B42Globals function displays the Unique Identifer, Location, and Module Template Path

## EXAMPLES

### EXAMPLE 1
```
Get-B42Globals
```
